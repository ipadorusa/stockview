## 8. 컴포넌트 목록

| 컴포넌트 | 사용 위치 | 설명 |
|----------|----------|------|
| `SearchBar` | Header | 종목 검색 + 자동완성 |
| `IndexCard` | 홈, 시장 | 지수 미니 카드 (차트 포함) |
| `StockRow` | 홈, 관심종목, 시장 | 종목 한 줄 요약 |
| `PriceDisplay` | 종목 상세 | 현재가 + 등락률 |
| `StockChart` | 종목 상세 | 캔들스틱/라인 차트 |
| `VolumeChart` | 종목 상세 | 거래량 차트 |
| `IndicatorToggle` | 종목 상세 | 기술 지표 ON/OFF |
| `NewsCard` | 홈, 뉴스, 종목 상세 | 뉴스 카드 |
| `ExchangeRate` | 홈, 종목 상세(미국) | 환율 표시 |
| `WatchlistButton` | 종목 상세 | ★ 관심종목 토글 |
| `TabNav` | 종목 상세 | 시세/차트/뉴스 탭 |
| `BottomTabBar` | 모바일 전체 | 하단 네비게이션 |

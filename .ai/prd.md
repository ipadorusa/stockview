# PRD: 주식 분석 웹 서비스 (StockView)

## 1. 개요

### 1.1 서비스 목적
초보 투자자가 한국/미국 주식 시세를 쉽게 조회하고, 차트로 분석하며, 관련 뉴스를 한눈에 확인할 수 있는 웹 서비스.

### 1.2 타겟 사용자
- **주 타겟**: 주식 투자를 시작한 지 1년 미만의 초보 투자자
- **특징**: 전문 용어에 익숙하지 않음, 모바일 사용 비중 높음, 간결하고 직관적인 UI 선호

### 1.3 핵심 가치
- **쉬운 시세 확인**: 복잡하지 않은 깔끔한 시세 화면
- **직관적 차트**: 초보자도 이해할 수 있는 차트와 지표 설명
- **한눈에 보는 뉴스**: 종목별/시장별 뉴스 큐레이션

## 2. 핵심 기능 (MVP)

### F1. 시세 조회
| 항목 | 설명 |
|------|------|
| 종목 검색 | 종목명/티커 검색 (한국: 삼성전자, 미국: AAPL) |
| 시세 정보 | 현재가, 전일 대비, 등락률, 거래량, 시가, 고가, 저가, 52주 최고/최저 |
| 시장 구분 | 한국(KOSPI/KOSDAQ), 미국(NYSE/NASDAQ) 탭 전환 |
| 환율 정보 | USD/KRW 환율 표시 (미국 주식 조회 시) |
| 장전/장후 데이터 | 프리마켓/애프터마켓 가격 표시 |
| 관심종목 | 로그인 사용자 관심종목 등록/관리 |

### F2. 차트
| 항목 | 설명 |
|------|------|
| 기간 선택 | 1주, 2주, 3주 (DB에 3주치만 보관. 1개월 이상은 Post-MVP) |
| 차트 유형 | 라인 차트, 캔들스틱 차트 |
| 기본 지표 | 이동평균선(MA 5/10), 거래량 (3주 데이터 기준) |
| 보조 지표 | RSI (기본 14일). MACD, 볼린저 밴드는 Post-MVP (데이터 부족) |
| 인터랙션 | 마우스 호버 시 해당 시점 데이터 표시, 줌/패닝 |

### F3. 뉴스
| 항목 | 설명 |
|------|------|
| 시장 뉴스 | 한국/미국 시장 주요 뉴스 피드 |
| 종목 뉴스 | 특정 종목 관련 뉴스 필터링 |
| 뉴스 요약 | 기사 제목 + 요약 + 출처 + 시간 |
| 감성 분석 | 뉴스 긍정/부정/중립 태그 (향후 확장) |

### F4. 회원 시스템
| 항목 | 설명 |
|------|------|
| 회원가입 | 이메일 + 비밀번호 기반 |
| 소셜 로그인 | Google, Kakao (MVP 이후 확장 가능) |
| 프로필 | 기본 정보 관리 |
| 관심종목 | 사용자별 관심종목 저장/관리 |

## 3. 페이지 구조 (Information Architecture)

```
/ (홈 - 대시보드)
├── /auth
│   ├── /login            — 로그인
│   ├── /register         — 회원가입
│   └── /forgot-password  — 비밀번호 찾기
├── /market
│   ├── /kr               — 한국 시장 개요 (KOSPI/KOSDAQ 지수, 등락 종목)
│   └── /us               — 미국 시장 개요 (S&P500/NASDAQ 지수, 등락 종목)
├── /stock
│   └── /[ticker]         — 종목 상세 (시세 + 차트 + 뉴스 통합)
├── /news
│   ├── /                 — 전체 뉴스 피드
│   └── /[category]       — 카테고리별 (한국/미국/산업별)
├── /watchlist            — 관심종목 (로그인 필요)
└── /settings             — 사용자 설정
```

## 4. 사용자 시나리오

### 시나리오 1: 종목 시세 확인
1. 사용자가 홈(대시보드)에 접속
2. 상단 검색바에 "삼성전자" 입력
3. 자동완성 드롭다운에서 "삼성전자 (005930)" 선택
4. `/stock/005930` 종목 상세 페이지로 이동
5. 현재가, 등락률, 거래량 등 시세 정보 확인
6. 하단에 관련 뉴스 목록 표시

### 시나리오 2: 차트 분석
1. 종목 상세 페이지에서 "차트" 탭 클릭
2. 기본 3주(3W) 캔들스틱 차트 표시
3. 기간을 "1주" / "2주"로 변경 가능
4. "이동평균선" 토글 ON → MA 5/10 오버레이
5. 지표 옆 (?) 아이콘 클릭 → 초보자용 설명 팝오버

### 시나리오 3: 미국 주식 조회
1. 검색바에 "AAPL" 입력
2. "Apple Inc. (AAPL)" 선택
3. 종목 상세 페이지에 USD 기준 시세 표시
4. 상단에 현재 환율(USD/KRW) 표시
5. 장전/장후 데이터가 별도 영역에 표시

### 시나리오 4: 관심종목 관리
1. 종목 상세 페이지에서 ★ 버튼 클릭
2. "관심종목에 추가되었습니다" 토스트 알림
3. `/watchlist` 페이지에서 관심종목 목록 확인
4. 각 종목의 현재가/등락률 한눈에 비교

### 시나리오 5: 뉴스 확인
1. 홈 대시보드 하단 "오늘의 뉴스" 섹션 확인
2. "더보기" 클릭 → `/news` 페이지 이동
3. "한국" / "미국" 탭으로 필터링
4. 뉴스 카드 클릭 → 원문 기사 링크(새 탭)

## 5. 비기능 요구사항

| 항목 | 요구사항 |
|------|----------|
| 성능 | 페이지 로딩 2초 이내 (LCP) |
| 반응형 | 모바일 (360px~) / 태블릿 (768px~) / 데스크탑 (1024px~) |
| SEO | Next.js SSR/SSG 활용, 메타태그 최적화 |
| 접근성 | WCAG 2.1 AA 수준 |
| 보안 | HTTPS, JWT 기반 인증, XSS/CSRF 방어 |
| 데이터 갱신 | 장중: 15분 지연 데이터 / 장전·장후: 별도 표시 |

## 6. 기술 스택

| 영역 | 기술 |
|------|------|
| Frontend | Next.js 15 + React 19 + TypeScript |
| UI | shadcn/ui + Tailwind CSS |
| 차트 | Lightweight Charts (TradingView) 또는 Recharts |
| 상태관리 | Zustand (클라이언트 상태) + TanStack Query (서버 상태) |
| Backend | Next.js API Routes (또는 별도 서버) |
| DB | PostgreSQL + Prisma ORM |
| 인증 | NextAuth.js (Auth.js) |
| 주식 데이터 | 한국: 한국투자증권 API, 미국: Yahoo Finance (비공식, fallback 검토 중) / Alpha Vantage (1일 25건 제한) |
| 배포 | Vercel (Frontend) + Railway/Supabase (DB) |

## 7. MVP 범위 및 우선순위

### MVP (Phase 1 출시)
1. ✅ 한국/미국 종목 검색 및 시세 조회
2. ✅ 기본 차트 (캔들스틱 + 라인)
3. ✅ 뉴스 피드 (시장별)
4. ✅ 이메일 회원가입/로그인
5. ✅ 관심종목

### Post-MVP (향후 확장)
- 소셜 로그인 (Google, Kakao)
- 뉴스 감성 분석
- 포트폴리오 관리 (매수/매도 기록)
- 알림 기능 (목표가 도달 시)
- 커뮤니티 / 토론 기능
- AI 기반 종목 분석 요약

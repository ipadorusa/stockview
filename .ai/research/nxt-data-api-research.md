# NXT(넥스트레이드) 데이터 API 리서치

**조사일**: 2026-04-03
**목적**: NXT 종가/거래 데이터를 API로 수집할 수 있는지 조사

---

## 1. NXT 공식 API (nextrade.co.kr)

**결론: 공개 API 없음**

- 넥스트레이드 공식 사이트(https://www.nextrade.co.kr/)는 시장정보 페이지에서 거래대금 상위종목, 프리마켓/메인마켓/애프터마켓 시세를 **웹 UI로만** 제공
- 개발자용 Open API, 데이터 피드, WebSocket 등 **외부 데이터 제공 서비스는 확인되지 않음**
- 사이트 구조에 API 문서, 개발자 센터 등의 메뉴 없음
- 시장정보 페이지의 데이터는 20분 지연 표시
- 넥스트레이드는 **거래소 운영사**이지 데이터 제공사가 아님 — 데이터 배포는 코스콤(KOSCOM) 등 유관기관 및 증권사를 통해 이루어지는 구조

## 2. KRX Open API (openapi.krx.co.kr / data.krx.co.kr)

**결론: 현재 NXT 데이터 미제공 (KRX 자체 데이터만)**

- KRX Open API 서비스 목록: 지수, 주식(유가증권/코스닥/코넥스), 증권상품, 채권, 파생상품, 일반상품, ESG
- 모든 서비스가 **KRX 거래 데이터만** 대상 (2010년 이후)
- NXT는 KRX와 별도 법인(다자간매매체결회사)이므로 KRX 데이터 마켓플레이스에서 NXT 시세를 제공하지 않음
- KRX와 NXT는 통합 시장감시 체계로 협력하지만, **데이터 제공은 별도**

## 3. 증권사 Open API (가장 유망한 경로)

### 한국투자증권 (KIS Developers)

**NXT 시세 조회 및 주문 모두 지원**

- REST API에서 `거래소ID구분코드` 파라미터 추가됨:
  - `KRX` — 한국거래소
  - `NXT` — 넥스트레이드
  - `SOR` — Smart Order Routing (최우선 호가 자동 선택)
- 국내주식 기본시세, 시세분석, 실시간시세 카테고리에서 NXT 데이터 조회 가능
- TR ID가 NXT 대응 버전으로 변경됨 (v1 → v2 등)
- **무료** (계좌 개설 필요, API 키 발급)
- 참고: https://apiportal.koreainvestment.com/

### 키움증권 REST API

**NXT 지원 확인됨**

- 2025년 3월 REST API 서비스 오픈과 함께 NXT 거래 지원
- NXT 매매체결대상종목 변경(편입/편출) 공지 활발히 운영 중
- 기존 Open API+ (COM 기반)도 NXT 대응 업데이트됨
- 참고: https://openapi.kiwoom.com/

### 대신증권 CYBOS Plus (사이보스플러스)

**부분 지원**

- `CpSvr7254` (매매입체분석): NXT 대응 개발 **없음** → KRX 데이터만 제공
- `StockChart` API: NXT 대응 개발 완료 → 거래소구분코드(통합/KRX/NXT) 입력 가능
- COM 기반이라 Windows + HTS 로그인 필요 (서버 환경 부적합)

## 4. 네이버 금융 (finance.naver.com)

**결론: NXT 시세 표시하지만 API 분리 불확실**

- 2025년 3월 4일부터 네이버페이 증권에서 NXT 시세 정보 제공 시작
- 종목별 상세화면에서 **KRX/NXT 시세를 함께 표시** 또는 **탭으로 구분**
- 네이버 통합검색, 모바일 경제판에도 NXT 시세 반영
- **그러나**: 네이버 금융의 기존 비공식 API(fchart, polling 등)에서 NXT 데이터가 어떻게 분리되는지는 **미확인**
- NXT 거래는 KRX 차트에 반영되지 않음 (별도 시장으로 취급)
- 지식iN 답변들에 따르면 "NXT는 시간외 단일가 거래일 뿐, 정규장 차트에 반영 안 됨"

## 5. 기타 데이터 소스

### pykrx (Python 라이브러리)

- KRX/Naver에서 스크래핑하는 라이브러리 — **NXT 대응 미확인**
- KRX 데이터만 대상이므로 NXT 데이터 수집 불가능할 가능성 높음

### FinanceDataReader

- 유사하게 KRX 기반 — NXT 별도 지원 여부 미확인

### KOSCOM (코스콤)

- 시장 데이터 인프라 운영사로 NXT 데이터도 취급할 가능성 있으나, 기관용/유료 서비스

## 종합 결론 및 권장사항

### 현재 상태 요약

| 데이터 소스 | NXT 데이터 제공 | 무료 | 서버 환경 사용 가능 |
|---|---|---|---|
| NXT 공식 사이트 | 웹 UI만 (API 없음) | - | - |
| KRX Open API | X (KRX만) | O | O |
| 한국투자증권 API | **O (시세+주문)** | O (계좌 필요) | **O** |
| 키움증권 REST API | **O** | O (계좌 필요) | **O** |
| 대신증권 CYBOS Plus | 부분적 | O | X (Windows) |
| 네이버 금융 | 표시함 (API 분리 불명) | - | 비공식 |
| pykrx | 미확인 (아마 X) | O | O |

### StockView 프로젝트 적용 시 권장 경로

1. **한국투자증권 KIS API** — 가장 현실적. REST API로 NXT 시세 별도 조회 가능. 거래소구분코드로 KRX/NXT 분리 가능. 단, 계좌 개설 + API 키 필요.
2. **네이버 금융 스크래핑** — 현재 프로젝트가 이미 네이버를 사용 중이므로, NXT 시세가 어떤 형태로 노출되는지 추가 조사 필요. fchart API에 거래소 구분 파라미터가 추가되었을 수 있음.
3. **넥스트레이드 웹 스크래핑** — 최후 수단. 공식 API가 없으므로 nextrade.co.kr의 시장정보 페이지를 스크래핑해야 하지만, 데이터가 20분 지연이고 법적 리스크 있음.

### 참고: NXT 거래시간

- 프리마켓: 08:00 ~ 08:50
- 메인마켓: 09:00:30 ~ 15:20
- 애프터마켓: 15:40 ~ 20:00 (공식, 기존 23:30에서 축소된 것으로 보임)
- 종가매매: 별도 운영

### 주의사항

- NXT 거래 가능 종목은 전체 상장종목이 아님 — 단계적 확대 중 (ETF 미포함 가능성)
- NXT 체결가는 다음날 KRX 시가에 직접 영향을 주지 않음
- NXT 점유율이 ~28.8%까지 성장 (2026년 3월 기준) — 데이터 중요도 증가 중

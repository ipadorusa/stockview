## 1. 현황 요약

### 데스크톱 (app-header.tsx)
```
[StockView] 홈 | 시장 | ETF | 스크리너 | AI 리포트 | 뉴스 | 게시판 | 관심종목  [검색] [테마] [로그인]
```
- 8개 링크가 동일 가중치로 수평 나열
- 배당, 실적, 섹터, 종목비교, 가이드 **미노출** (5개 페이지)

### 모바일 BottomTabBar (bottom-tab-bar.tsx)
```
홈 | 시장 | 뉴스 | MY   (4탭)
```
- ETF, 스크리너, AI 리포트, 게시판, 관심종목 접근 불가
- `/screener`, `/reports` 등 방문 시 탭 활성 표시 없음

### 모바일 Sheet (햄버거)
- 데스크톱과 동일 8개 navLinks + 로그인/회원가입
- 그룹핑 없이 수직 나열
- 배당/실적/섹터/비교/가이드 역시 미포함

## 2. 데스크톱 메가메뉴 설계

### 2.1 1단 네비게이션 (5개 카테고리)

```
[StockView]  홈  투자 ▼  분석 ▼  뉴스  더보기 ▼     [검색] [테마] [로그인]
```

### 2.2 드롭다운 구조

**투자 드롭다운:**
```
┌─────────────────────────────────┐
│  시장                            │
│  ├─ 시장 개요        /market     │
│  ├─ ETF             /etf        │
│  └─ 섹터별 종목     /sectors    │
│                                  │
│  캘린더                          │
│  ├─ 배당 캘린더     /dividends  │
│  └─ 실적 캘린더     /earnings   │
└─────────────────────────────────┘
```

**분석 드롭다운:**
```
┌─────────────────────────────────┐
│  종목 분석                       │
│  ├─ 스크리너        /screener   │
│  ├─ AI 리포트       /reports    │
│  └─ 종목 비교       /compare    │
│                                  │
│  학습                            │
│  └─ 투자 가이드     /guide      │
└─────────────────────────────────┘
```

**더보기 드롭다운:**
```
┌─────────────────────────────────┐
│  ├─ 게시판          /board      │
│  ├─ 관심종목        /watchlist  │
│  ├─ 서비스 소개     /about      │
│  └─ 문의하기        /contact    │
└─────────────────────────────────┘
```

### 2.3 활성 상태 로직

| 1단 메뉴 | 활성 조건 (pathname) |
|----------|---------------------|
| 홈 | `=== "/"` |
| 투자 | `.startsWith("/market")` \|\| `/etf` \|\| `/sectors` \|\| `/dividends` \|\| `/earnings` |
| 분석 | `.startsWith("/screener")` \|\| `/reports` \|\| `/compare` \|\| `/guide` \|\| `/stock/` |
| 뉴스 | `.startsWith("/news")` |
| 더보기 | `/board` \|\| `/watchlist` \|\| `/about` \|\| `/contact` |

### 2.4 구현 방식

- 드롭다운: Radix UI `NavigationMenu` 사용 (shadcn/ui에 포함)
- 호버 시 열림, 클릭 시 이동
- 각 드롭다운 최대 너비 320px, 섹션 헤더는 `text-xs text-muted-foreground uppercase`

## 3. 모바일 5탭 BottomTabBar 설계

### 3.1 탭 구성

```
┌─────┬─────┬─────┬─────┬─────┐
│ 🏠  │ 📈  │ 🔍  │ 📰  │ 👤  │
│ 홈  │ 투자 │ 분석 │ 뉴스 │ MY  │
└─────┴─────┴─────┴─────┴─────┘
```

| 탭 | 아이콘 | 이동 경로 | 활성 조건 |
|----|--------|----------|----------|
| 홈 | `Home` | `/` | `=== "/"` |
| 투자 | `TrendingUp` | `/market` | `/market` \|\| `/etf` \|\| `/sectors` \|\| `/dividends` \|\| `/earnings` |
| 분석 | `BarChart3` | `/screener` | `/screener` \|\| `/reports` \|\| `/compare` \|\| `/guide` \|\| `/stock/` |
| 뉴스 | `Newspaper` | `/news` | `/news` |
| MY | `User` | `/mypage` | `/mypage` \|\| `/watchlist` \|\| `/settings` |

### 3.2 투자/분석 탭 하위 접근

**방안 A (권장): 각 탭 랜딩 페이지에 퀵 링크 카드**

투자 탭(`/market`) 상단:
```
┌────────┐ ┌────────┐ ┌────────┐ ┌────────┐
│  ETF   │ │ 섹터   │ │ 배당   │ │ 실적   │
└────────┘ └────────┘ └────────┘ └────────┘
```

분석 탭(`/screener`) 상단:
```
┌────────┐ ┌────────┐ ┌────────┐ ┌────────┐
│스크리너│ │AI리포트│ │종목비교│ │  가이드 │
└────────┘ └────────┘ └────────┘ └────────┘
```

**방안 B: 롱프레스 시 ActionSheet**
- 탭 롱프레스 → 하위 메뉴 바텀시트 표시
- 추가 구현 복잡도가 높아 Phase 2로 미룸

### 3.3 코드 변경

```tsx
// bottom-tab-bar.tsx 변경안
const tabs = [
  { href: "/", label: "홈", icon: Home,
    isActive: (p: string) => p === "/" },
  { href: "/market", label: "투자", icon: TrendingUp,
    isActive: (p: string) => ["/market","/etf","/sectors","/dividends","/earnings"].some(x => p.startsWith(x)) },
  { href: "/screener", label: "분석", icon: BarChart3,
    isActive: (p: string) => ["/screener","/reports","/compare","/guide","/stock/"].some(x => p.startsWith(x)) },
  { href: "/news", label: "뉴스", icon: Newspaper,
    isActive: (p: string) => p.startsWith("/news") },
  { href: "/mypage", label: "MY", icon: User,
    isActive: (p: string) => ["/mypage","/watchlist","/settings"].some(x => p.startsWith(x)) },
]
```
